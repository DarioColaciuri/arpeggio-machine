## Features

- **53 chord types** across 5 categories (Basic, Seventh, Sixth, Extended, Altered)
- **Realistic 24-fret fretboard** with rosewood texture, wound strings, fret markers, and nut
- **Piano-style note selector** with sharp/flat notation per key
- **Audio playback** via Web Audio API — arpeggiates chord tones
- **Root note highlighting** with distinct color indicator
- **Collapsible chord categories** with color-coded dots
- **Fully keyboard accessible** with ARIA labels, skip link, and screen reader support

## Getting Started

```bash
npm install
npm run dev
```
